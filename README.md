# CodeBuddy-QA-Context

> CodeBuddy 质量保证上下文存储仓库

## 概述

本仓库用于存储和管理 CodeBuddy 项目相关的质量保证（QA）上下文信息、文档、配置和测试数据。

## 项目结构

```
.
├── README.md           # 项目说明文档
├── .gitignore          # Git 忽略文件配置
├── docs/               # 文档目录
│   ├── qa-guidelines.md      # QA 指南
│   ├── testing-strategy.md   # 测试策略
│   └── best-practices.md     # 最佳实践
├── configs/            # 配置文件目录
│   ├── test-config.yaml      # 测试配置
│   └── qa-standards.json     # QA 标准定义
└── templates/          # 模板文件目录
    ├── test-report.md        # 测试报告模板
    └── issue-template.md     # 问题报告模板
```

## 用途

- **上下文管理**: 存储和维护 QA 相关的上下文信息
- **文档共享**: 提供 QA 团队的文档和指南
- **配置统一**: 管理测试和质量标准的配置文件
- **模板复用**: 提供标准化的测试报告和问题模板
